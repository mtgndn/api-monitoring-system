# 🚀 API Monitoring System

A full-stack API monitoring dashboard with API key authentication.

## 🔧 Tech Stack

* Backend: Node.js, Express
* Database: SQLite
* Frontend: Next.js, TailwindCSS
* Charts: Recharts

## ✨ Features

* 🔐 API Key Authentication
* 📊 Analytics Dashboard (daily usage, top endpoints)
* 🔄 API Key Regeneration
* 🚀 Protected Routes (middleware)
* ⚡ Rate Limiting

## 🚀 Getting Started

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

## 🔑 How It Works

1. Generate API key from Settings
2. Use API key in headers:

```
x-api-key: YOUR_API_KEY
```

3. Track usage via dashboard
